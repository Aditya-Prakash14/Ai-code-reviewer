## 🚀 Quick Start

### 1. Backend Setup
```bash
cd Backend
npm install
npm run dev  # Starts on localhost:3000
```

### 2. Frontend Setup  
```bash
cd frontend
npm install
npm run dev  # Starts on localhost:5173
```

### 3. Environment Configuration
```bash
# Create Backend/.env
GOOGLE_GEMINI_KEY=your_google_ai_api_key
```
